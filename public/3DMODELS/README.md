# Modelos 3D

Esta pasta contém modelos 3D utilizados pelo projeto.

## Arquivos grandes

Os arquivos .glb dos modelos 3D não são armazenados no GitHub devido ao seu tamanho. Estes arquivos devem ser armazenados separadamente num serviço de armazenamento como o Firebase Storage.

## Modelos disponíveis

O projeto utiliza os seguintes modelos:
- IQOS_ILUMA_I_BREEZE.glb
- IQOS_ILUMA_I_ONE_BREEZE.glb
- IQOS_ILUMA_I_PRIME_BREEZE.glb
